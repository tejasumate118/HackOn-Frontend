# HackOn-Frontend


## 📋 Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn
- Modern web browser

## 🔧 Installation

1. Clone the repository:
```bash
git clone https://github.com/anii-123/Hackon_FourLoop
cd frontend
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

## 🛠️ Built With

- React.js
- TypeScript
- Material-UI
- Redux
